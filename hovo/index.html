Ի՞նչ է GitHub-ը

Բարձր մակարդակով GitHub-ը ծառայություն է, որն օգնում է մշակողներին պահել և կառավարել իրենց կոդը, ինչպես նաև հետևել և վերահսկել իրենց կոդի փոփոխությունները:GitHub-ը շահույթ հետապնդող ընկերություն է, որն առաջարկում է ամպի վրա հիմնված Git պահեստի հոստինգ ծառայություն: Ըստ էության, դա շատ ավելի հեշտ է դարձնում անհատների և թիմերի համար Git-ի օգտագործումը տարբերակների վերահսկման և համագործակցության համար:

Git Hub

Git Hub-ում փոփոխություներ անելու մասին

Git Hub-ի ռեպոն համակարգչում ունենալու համար,terminal-ով մտնում ենք մեր ուզած տեղը և նախապես ռեպոի պատճենած HTTP-ն git clone հրամանով տեղադրում այնտեղ։

Նոր branch ստեղծելու համար,մտնում ենք մեր ռեպոն։Git checkout հրամանով տեղափոխվում develop,ապա git fetch հրամանով իմանում թե ինչ branch կա և git pull։Git -b հրամանով ստեղծում նոր branch:Git add . հրամանով իմանում եղած կատալոգը,git status հրամանով ինչ ավելացավ,git commit -m հրամանով ավելացնում և git push --set-upstream origin հրամանով այն հրում ռեպո։
