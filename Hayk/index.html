HTML lang

HTML lang հատկանիշն օգտագործվում է համացանցում տեքստային բովանդակության լեզուն նույնականացնելու համար: Այս տեղեկատվությունը օգնում է որոնման համակարգերին վերադարձնել որոշակի լեզվի արդյունքներ, և այն նաև օգտագործվում է էկրանի ընթերցողների կողմից, որոնք փոխում են լեզվական պրոֆիլները՝ ճիշտ շեշտադրում և արտասանություն ապահովելու համար: Փաստաթղթի հիմնական լեզուն սահմանելու համար դուք օգտագործում եք langտարրի հատկանիշը html.
Եթե ​​ձեր փաստաթուղթը HTML է (այսինքն՝ վերլուծված է որպես text/html), օգտագործեք langհատկանիշը՝ փաստաթղթի լեզուն կամ տեքստի բլոկի լեզուն սահմանելու համար: Օրինակ, հետևյալ կոդը կսահմանի լռելյայն լեզուն ֆրանսերենին.
