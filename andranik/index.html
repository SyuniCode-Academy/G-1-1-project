Ինչպես ձեզ տրամադրված github-ի ռեպոն համակարգչում ունենալ ?

Մեզ տրամադրված github-ի ռեպոն համակարգչում ունենալու համար պետք է copy անել https կամ ssh կոդը ռեպոզիտորիաից և տերմինալի օգնությամբ գրել git remote add origin և տեղադրել copy արած հղումը այնուհետև գրում ենք git fetch հետո git pull որից հետո մենք կունենանք github-ի ռեպոն մեր համակարգչում

Ինչպես github-ի ռեպո–ում առաջադրանք կատարել և այդ փոփոխությունները տեղադրել github-ում?

Github-ի ռեպո–ում առաջադրանքը կատարելուց հետո փոփոխությունները տեղադրելու համար պետք է սկզբում գրել git add (ֆայլի անունը) հԵտո գրել git commit -m(message) (ինչ փոփոխություններ է կատարվել)իսկ այդ ամենը որպեսզի ուղղարկենք github պետք է գրել git push -u origin (ձեր branch-ի անունը):
